竞彩混合过关的解释说明

智赢2串1
根据国家体育总局体育彩票管理中心的公告， 竞彩足篮球混合过关投注方式将于2012年12月18日面市。这种新颖的投注方式究竟带来什么全新的乐趣。请按以下详细了解：

无效比赛处理方式（单关）

由于比赛取消、延期等原因，导致某场比赛被认定为无效场次，则该场比赛的所有单场投注按退票处理。


混合过关是什么？怎么才算是混合过关？

“混合过关”并非玩法，而是一种投注方式。在我们以往的竞彩过关投注中，全部是前后比赛场次选择同一运动项目的同一种游戏进行投注，这种投注方式被称为“一般过关投注”，而现在新的投注方式出现了：前后比赛场次选择同一运动项目不同游戏也可以进行投注，这种投注方式称为“混合过关投注”。


足球和篮球的不同游戏玩法是否能“混合过关”？

不同运动项目的是不能混合过关投注的，所以竞彩足球和竞彩篮球是无法混合一切过关投注的。


同一场比赛的不同玩法能否“混合过关”？

是不可以的。第一是规则上不允许。第二是，选择相同比赛的不同玩法不能称为过关，因为比分“0-0”打出后，半全场、总进球、胜平负是必然结果，根本不需要过关。我们应该更好的理解一下“过关”的含义，选择不同比赛的选项，才是过关。


混合过关每注所能达到的最高关数？

简单来说是按照你的混合过关投注内容上限最低的过关为标准，竞彩中过关数最高的是胜平负(8关)，最低是比分(4关)，当您的混合过关的投注内容里面含有1个比分的投注选项，那么你这个方案就最高只能按照4关来投注。有些类似“木桶原理”，木桶能承装多少水（能串多少关），与最短的那块木板有关（与所选到的玩法中，关数上限最低的那个玩法一致）。

目前竞彩足球和竞彩篮球的最高关数可以参考一下：

足球：胜平负8关 / 总进球6关 / 半全场4关 / 比分4关

篮球：胜负8关 / 让分胜负8关 / 大小分8关 / 胜分差4关


“混合过关”每注最大中奖金额？

每注能够获得的最大中奖金额，与过关场次数有直接关系，这个规则与以往的“一般过关投注”奖金上限相同，如下：

2场或3场过关投注，单注最高奖金限额20万元。

4场或5场过关投注，单注最高奖金限额50万元。

6场及6场以上过关投注，单注最高奖金限额100万元。


“冠军、冠亚军”这种名次玩法是否可以参与“混合过关”？

不可以。在《通知》和《公告》中明确指出，是单场竞猜游戏增加混合过关投注方式。另外，在“冠军、冠亚军”玩法规则中明确指出不能进行过关投注。
